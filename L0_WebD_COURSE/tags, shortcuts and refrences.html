tags and attributes

  1. ! for complete html skeleton
  2. meta tag for description, keywords, robot
  3. link rel="stylesheet" href="aryan.css"
  4. script src="aryan.js"
  5. p tag for para
  6. br to break a line
  7. strong to bolden
  8. em to italicise
  9. hr for making a horizontal ruled line
  10. h for heading
  11. loremx print text x dummy words
  12. img for adding image
  13. a href for hyperlinks
  14. ul & ol & li listts

shortcut keys

  • shift + alt + down arrow to copy selected text in next line
  • alt+z to toggle wrap tex
  • ctrl + / for converting some text to comment
  • ctrl+ enter to go to direct jump line
